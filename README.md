# Tic Tac Toe

<p align="center"><img src="./thumbnail.png"></p>

Permainan Tic Tac Toe dengan algoritme sederhana.

## Instalasi

```
git clone https://github.com/kangzoel/simple-tictactoe.git
cd simple-tictactoe
yarn
```

Development server: `yarn dev`
